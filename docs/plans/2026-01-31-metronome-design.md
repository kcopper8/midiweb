# 메트로놈 (리듬 가이드) 기능 설계

## 개요

연주에 도움이 되는 메트로놈 기능입니다. 일정한 박자를 제공하여 템포를 맞추며 연주할 수 있습니다.

## 핵심 동작

### 메트로놈 기능
- 시작/정지 버튼으로 on/off
- 클릭음 사용 (강박: 높은 음, 약박: 낮은 음)

### 박자 지원

| 박자 | 패턴 |
|------|------|
| 4/4 | 똑-틱-틱-틱 (강-약-약-약) |
| 3/4 | 똑-틱-틱 (강-약-약) |

### 템포 설정
- 범위: 40~200 BPM
- 프리셋 버튼: 60, 80, 100, 120 BPM
- 슬라이더: 세밀한 조절
- 기본값: 100 BPM

### 소리 생성
- Web Audio API의 OscillatorNode 사용
- 강박: 1000Hz, 약박: 800Hz
- 짧은 클릭음 (0.05초)

## UI 레이아웃

### 상단 컨트롤 영역에 새 행 추가

```
┌─────────────────────────────────────────────────────┐
│  스케일  [Major] [Minor] [Pentatonic] ...           │
│  옥타브  [1] [2] [3]   파형  [Sine] [Square] ...    │
│  메트로놈 [▶] 박자 [4/4] [3/4]  BPM [60][80][100][120] ───○─── 100 │
└─────────────────────────────────────────────────────┘
```

### 컨트롤 구성
- 시작/정지 버튼: ▶ / ⏹ 토글
- 박자 선택: 4/4, 3/4 토글 버튼
- BPM 프리셋: 60, 80, 100, 120 버튼
- BPM 슬라이더: 40~200 범위, 현재 값 표시

### 반응형 처리
- 모바일에서는 메트로놈 행이 두 줄로 줄바꿈 가능
- 기존 컨트롤 스타일과 동일하게 유지

## 향후 확장

이 기능은 배경 반주 기능의 기반이 됩니다:
- **드럼/비트**: 메트로놈 템포 시스템 위에 드럼 패턴 추가
- **화음 반주**: 템포에 맞춘 코드 진행 제공
